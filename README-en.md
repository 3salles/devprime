## Technology
This project use JavaScript, React and Node.

## Installation
### Pre Requirements
The following tools are needed to: [Git](https://git-scm.com), [Node.js](https://nodejs.org/en/) and ```npm```(package manager that comes with Node.js).

### Install Dependencies
To install the dependencies, follow these steps:

```bash
# Clone this repository
$ git clone <https://github.com/3salles/devprime>

# Acess the project's folder 
$ cd devprime

# Install dependencies
$ npm install
```
## How to Use
With the dependencies installed, use the following command inside the folder.
```bash
npm install

# Acess the link in port 5000
<http://localhost:5000>
```
